About:

This is a rails inspired go framework that uses Hacker News style article submission, comments, and voting.

The app you are using is built with the feedback framework and you can make your own app with it.

This sample app is open source https://github.com/andrewarrow/remoterenters/

RemoteRenters is a place to discuss the future of working remote from home. But it is also a place like reddit where you can make a "sub remote renter" topic and be the mod of that sub. Just goto any slash rr topic /rr/topic like:

  • /rr/politics
  • /rr/news
  • /rr/your-new-topic-name-here
